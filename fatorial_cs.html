Exercício: Cálculo do Fatorial

Escreva um programa que solicite ao usuário que digite um número. Calcule o fatorial desse número e exiba o resultado no console.

Variáveis

  • A variável fatorial armazena o resultado do fatorial.
  • A variável numero armazena o número inserido pelo usuário.

Entrada de Dados

  • O programa utiliza Console.ReadLine() para receber a entrada do usuário.
  • O valor inserido pelo usuário é convertido em um número inteiro usando int.Parse().

Loop (while)

  • O programa utiliza um loop while para calcular o fatorial.
  • A variável i é inicializada com o valor do número.
  • O loop continua enquanto i for maior ou igual a 1.
  • Em cada iteração, o valor do fatorial é atualizado multiplicando-se pela variável i, e o valor de i é decrementado.

Saída de Dados (Console.WriteLine())

  • Após o loop, o programa exibe o resultado do fatorial usando Console.WriteLine().

using System;

class Program
{
    public static void Main(string[] args)
    {
        BigInteger fatorial = 1;

        Console.WriteLine("Insira um número: ");
        int numero = int.Parse(Console.ReadLine());

        int i = numero;
        while (i >= 1) 
        { 
            fatorial *= i; 
            i--; 
        }

        string escreva = "O fatorial de " + numero + " é " + fatorial;
        Console.WriteLine(escreva);
    }
}
