Exercício: Contagem de Números Pares e Ímpares

Escreva um programa que conte e exiba a quantidade de números pares e ímpares no intervalo de 1 a 100.

Variáveis

  • A variável par armazena a quantidade de números pares.
  • A variável impar armazena a quantidade de números ímpares.
  • A variável numero é usada para iterar de 1 a 100.

Loop (while)

  • O programa utiliza um loop while para iterar de 1 a 100.
  • Enquanto o valor da variável numero for menor ou igual a 100, o loop continuará.

Condição IF

  • Dentro do loop, o programa verifica se o número é divisível por 2 (par).
  • Se a condição for verdadeira, a variável par é incrementada e o número é exibido como "Número par: [numero]".
  • Se a condição for falsa, a variável impar é incrementada e o número é exibido como "Número ímpar: [numero]".

Saída de Dados (Console.WriteLine())

  • Após o loop, o programa exibe a quantidade de números pares e ímpares.

    using System;

    class Program
    {
        public static void Main(string[] args)
        {
            int par = 0;
            int impar = 0;
            int numero = 1;

            while (numero <= 100)
            {
                if (numero % 2 == 0)
                {
                    par++;
                    Console.WriteLine("Número par: " + numero);
                }
                else
                {
                    impar++;
                    Console.WriteLine("Número ímpar: " + numero);
                }
                numero++;
            }

            Console.WriteLine("A contagem de números pares entre 1 e 100 é: " + par);
            Console.WriteLine("A contagem de números ímpares entre 1 e 100 é: " + impar);
        }
    }
