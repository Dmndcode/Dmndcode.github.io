Exercício: Encontrar o Maior Número em uma Série

Escreva um programa que solicita ao usuário uma série de números separados por vírgula e encontra o maior número nessa série.

Variáveis

  • A variável input armazena a entrada do usuário.
  • A variável numbers é um array que armazena os números separados por vírgula.
  • A variável max armazena o maior número encontrado.

Entrada de Dados (Console.ReadLine())

  • O programa solicita ao usuário que insira uma série de números separados por vírgula.
  • A entrada do usuário é armazenada na variável input.

Manipulação de Strings e Array

  • O programa utiliza o método Split() para separar a string de entrada em um array de números.
  • O método Trim() é utilizado para remover quaisquer espaços em branco em torno de cada número.
  • Se a entrada do usuário for nula, é atribuída uma string vazia à variável input.
  • Se o array de números for nulo, é atribuído um array vazio à variável numbers.

Loop (foreach)

  • O programa utiliza um loop foreach para iterar sobre cada número no array numbers.
  • Para cada número, o programa verifica se é possível convertê-lo para um inteiro usando o método TryParse().
  • Se a conversão for bem-sucedida, o programa compara o número atual com o valor máximo encontrado até o momento e atualiza a variável max se necessário.
  • Se a conversão falhar, o programa exibe uma mensagem de entrada inválida e interrompe o loop usando o comando break.

Saída de Dados (Console.WriteLine())

  • Após o loop, o programa exibe o maior número encontrado usando a variável max.

Código em C#


using System;

class Program
{
    public static void Main(string[] args)
    {
        Console.WriteLine("Escreva uma série de números separados por vírgula");
        string input = Console.ReadLine() ?? string.Empty;
        String[]? numbers = input?.Split(',');
        int max = int.MinValue;

        foreach (String number in numbers ?? Array.Empty())
        {
            if (int.TryParse(number?.Trim(), out int currentNmbr))
            {
                if (currentNmbr > max) { max = currentNmbr; }
            }
            else { Console.WriteLine("Entrada inválida: " + number); break; }
        }

        Console.WriteLine("O maior número da série é: " + max);
    }
}
