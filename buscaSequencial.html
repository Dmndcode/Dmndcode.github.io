Exercício de Busca Sequencial em C#

Neste exercício, vamos explorar o conceito de busca sequencial em C#. A busca sequencial é um algoritmo simples usado para encontrar um elemento específico em uma coleção de dados, como um vetor ou uma lista. Neste caso, usaremos uma lista de tipos de café como exemplo.

Variáveis Utilizadas

  • tiposDeCafes: Uma lista de objetos representando diferentes tipos de café.
  • nomeBuscado: O termo de busca inserido pelo usuário.
  • encontrado: Uma variável booleana para controlar se o tipo de café foi encontrado.

Funcionalidade de Busca Sequencial

  • A função BuscaCafe verifica se o nome ou a descrição de um café contém o termo de busca, após remover acentos e tornar tudo em letras minúsculas.
  • O resultado é usado para determinar se um tipo de café corresponde ao termo de busca.

Processo Principal

  1. Uma lista de objetos de tipos de café é criada, contendo diferentes tipos de café com nomes e descrições.
  2. O usuário é solicitado a inserir um termo de busca (por exemplo, "arabica").
  3. Para cada tipo de café na lista:
    • A função de busca é chamada para verificar se o termo de busca corresponde ao nome ou descrição do café.
    • Se uma correspondência for encontrada, o nome do café e sua descrição são exibidos.
    • A variável encontrado é definida como verdadeira.
  4. Se nenhum tipo de café corresponder ao termo de busca, uma mensagem "Tipo de café não encontrado" é exibida.

Resultado

  • Se um tipo de café correspondente for encontrado, suas informações serão exibidas na saída.
  • Se nenhum tipo de café corresponder, uma mensagem indicando que o tipo de café não foi encontrado será exibida.

Código Completo


            using buscaSequencial.Models;
using buscaSequencial.Helpers;
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
using System.Threading.Tasks;


namespace buscaSequencial
{
    class Program
    {
        static bool BuscaCafe(TipoDeCafe cafe, string termoBusca)
        {
            string nomeSemAcentos = StringHelper.RemoverAcentos(cafe.Nome.ToLower());
            string termoSemAcentos = StringHelper.RemoverAcentos(termoBusca.ToLower());

            return nomeSemAcentos.Contains(termoSemAcentos);
        }

        static void Main(string[] args) 
        {
            List tiposDeCafes = new List {
                new TipoDeCafe { Nome = "Café Arábica", Descricao = "Um café de sabor suave e aromático, cultivado em regiões de alta altitude." },
                new TipoDeCafe { Nome = "Café Robusta", Descricao = "Conhecido por sua robustez e sabor intenso, frequentemente usado em blends de espresso." },
                new TipoDeCafe { Nome = "Café Kona", Descricao = "Originário do Havaí, este café possui notas frutadas e é cultivado em solos vulcânicos." },
                new TipoDeCafe { Nome = "Café Geisha", Descricao = "Uma variedade de café especial conhecida por seus sabores florais e complexidade." }
            };

            Console.Write("Insira o nome de um tipo de café: ");
            string nomeBuscado = Console.ReadLine().Trim().ToLower();

            bool encontrado = false;
            foreach (var cafe in tiposDeCafes)
            {
                if (BuscaCafe(cafe, nomeBuscado))
                {
                    Console.WriteLine($"Tipo de Café: {cafe.Nome}");
                    Console.WriteLine($"Descrição: {cafe.Descricao}");
                    encontrado = true;
                }
            }
            if (!encontrado)
            {
                Console.WriteLine("Tipo de café não encontrado.");
            }
        }
    }
}
