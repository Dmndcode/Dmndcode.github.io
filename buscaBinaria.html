Exercício de Busca Binária em C#

Neste exercício, vamos explorar o conceito de busca binária em C#. A busca binária é um algoritmo eficiente para encontrar um elemento específico em uma lista ordenada. Ela funciona dividindo repetidamente a lista ao meio e comparando o elemento do meio com o elemento desejado.

Funcionalidade da Busca Binária

A função de busca binária verifica se o elemento do meio da lista é igual ao elemento desejado. Se for, o elemento é encontrado e sua posição é retornada. Caso contrário, se o elemento do meio for menor que o elemento desejado, a busca é realizada na metade direita da lista. Se for maior, a busca é realizada na metade esquerda da lista. Esse processo é repetido até que o elemento seja encontrado ou o intervalo de busca seja reduzido a zero.

Algoritmo da Busca Binária

        function BuscaBinaria(array, alvo):
            esquerda = 0
            direita = tamanho do array - 1
            
            enquanto esquerda <= direita:
                meio = esquerda + (direita - esquerda) / 2
                
                se array[meio] for igual a alvo:
                    retorne meio
                
                se array[meio] for menor que alvo:
                    esquerda = meio + 1
                senão:
                    direita = meio - 1
            
            retorne -1

Exemplo de Uso

        using System;
        using buscaBinaria.Helpers;
        using buscaBinaria.Models;

        namespace buscaBinaria
        {
            class BinarySearch
            {
                static void Main(string[] args)
                {
                    int[] sortedArray = { 2, 5, 8, 12, 16, 23, 38, 56, 72, 91 };

                    ArrayModel arrayModel = new ArrayModel(sortedArray);

                    int target = 72;

                    if (BoundsHelper.IsOutsideBounds(arrayModel.Data, target))
                    {
                        Console.WriteLine($"O elemento {target} está fora dos limites do array.");
                    }
                    else
                    {
                        int resultIndex = ArrayHelper.BinarySearch(arrayModel.Data, target);

                        if (resultIndex != -1)
                            Console.WriteLine($"Valor {target} encontrado no índice {resultIndex}");
                        else
                            Console.WriteLine($"Valor {target} não encontrado na lista");

                    }

                    Console.ReadLine();
                }
            }
        }

Este é um exemplo básico de como implementar e usar a busca binária em C#. Lembre-se de que a busca binária funciona apenas em listas ordenadas.
