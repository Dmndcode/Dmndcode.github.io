Exercicio:

Escreva um programa que solicite continuamente ao usuário que digite um número ou "ok" para sair. Calcule a soma de todos os números previamente inseridos e exiba o resultado no console.

Variáveis

  • A variável sum armazena a soma dos números inseridos.
  • A variável multiply armazena a multiplicação dos números.
  • A variável divide armazena o resultado da divisão.

Entrada de Dados

  • O programa utiliza Console.ReadLine() para receber a entrada do usuário.
  • O valor inserido pelo usuário é armazenado na variável input como uma string.

Estrutura de Repetição (while loop)

  • O programa utiliza um loop while para permitir que o usuário insira vários números ou digite "ok" para sair.
  • O loop se repete até que a condição while (true) seja falsa ou até que o comando break seja executado.

Conversão de String para Inteiro (int.TryParse())

  • O programa utiliza int.TryParse(input, out number) para converter a string input em um número inteiro.
  • A função int.TryParse() tenta converter a string em um número inteiro e retorna true se for bem-sucedida.
  • Se a conversão falhar, retorna false.

Condição IF

  • O programa utiliza a estrutura de controle if para realizar diferentes ações com base nas condições.
  • Verifica-se se a entrada é igual a "ok" para sair do loop.
  • Em seguida, verifica-se se o número inserido é diferente de zero antes de realizar os cálculos.
  • Se a condição for atendida, a soma e a multiplicação são atualizadas com o valor do número inserido.

Divisão com Precisão Decimal

  • Para realizar a divisão com precisão decimal, o programa converte a variável multiply para o tipo decimal.
  • Em seguida, realiza a divisão pelo valor da variável sum.
  • O resultado é armazenado na variável divide como (decimal)multiply / sum.

Saída de Dados (Console.WriteLine())

  • O programa utiliza Console.WriteLine() para exibir os resultados parciais e finais dos cálculos.
  • Os resultados da soma, multiplicação e divisão são exibidos no console.

    using System;
    
    class Program
    {
        public static void Main(string[] args)
        {
            int sum = 0;
            int multiply = 1;
            decimal divide = 0;
            string input;
    
            while (true)
            {
                Console.WriteLine("Insira um número ou digite 'ok' para sair: ");
                input = Console.ReadLine();
    
                if (input.ToLower() == "ok")
                {
                    break;
                }
    
                int number;
                if (int.TryParse(input, out number) && number != 0)
                {
                    sum += number;
                    multiply *= number;
                    divide = (decimal)multiply / sum;
                }
                else
                {
                    if (number == 0)
                    {
                        Console.WriteLine("Uma divisão por zero não possui um valor numérico válido.");
                    }
                    else
                    {
                        Console.WriteLine("Valor Inválido, Por gentileza insira um número ou digite 'ok'. ");
                    }
                }
            }
    
            Console.WriteLine("A soma dos valores é: " + sum);
            Console.WriteLine("A multiplicação dos valores é: " + multiply);
            Console.WriteLine("A divisão do total de soma por total de multiplicação é: " + divide);
        }
    }
