Jogo de Adivinhação

Este jogo é um programa simples em C# que permite ao jogador adivinhar um número secreto entre 1 e 7. O jogador tem um número limitado de tentativas para acertar o número. Se o jogador adivinhar corretamente, ele ganha o jogo; caso contrário, perde.

Variáveis

  • A variável numSecreto armazena o número secreto gerado aleatoriamente.
  • A variável tentativas armazena o número de tentativas disponíveis para o jogador.
  • A variável palpite armazena a entrada do jogador, ou seja, o número que ele está adivinhando.

Geração de Número Aleatório (Random)

  • O programa utiliza a classe Random para gerar um número aleatório entre 1 e 7, que é o número secreto a ser adivinhado.
  • Isso é feito através da criação de uma instância da classe Random e do uso do método Next() para gerar um número aleatório.

Loop (while)

  • O programa utiliza um loop while para permitir que o jogador faça várias tentativas.
  • O loop continuará executando enquanto o número de tentativas for maior que 0.

Entrada do Jogador (Console.ReadLine())

  • Dentro do loop, o programa solicita ao jogador que digite um número entre 1 e 7.
  • O número digitado pelo jogador é armazenado na variável palpite após ser convertido para um tipo int usando o método int.Parse().

Condição IF

  • Após o jogador fazer um palpite, o programa verifica se o palpite é igual ao numSecreto.
  • Se a condição for verdadeira, ou seja, o jogador acertar o número secreto, o programa exibe a mensagem "Você acertou, Parabéns!" e o jogo é encerrado usando a declaração return.
  • Se a condição for falsa, ou seja, o jogador errar o número secreto, o número de tentativas é decrementado e o programa exibe a mensagem "Você errou! Tentativas restantes: [tentativas]".

Saída de Dados (Console.WriteLine())

  • Após o loop, se o número de tentativas chegar a 0, o programa exibe a mensagem "Você perdeu! O número secreto era: [numSecreto]".

using System;

class Program
{
    public static void Main(string[] args)
    {
        Random rnd = new Random();
        int numSecreto = rnd.Next(1, 7);
        int tentativas = rnd.Next(2, 4);
        Console.WriteLine("Número secreto: " + numSecreto); // Apenas para teste

        while (tentativas > 0)
        {
            Console.WriteLine("Digite um número entre 1 e 7: ");
            int palpite = int.Parse(Console.ReadLine());

            if (palpite == numSecreto)
            {
                Console.WriteLine("Você acertou, Parabéns!");
                return;
            }
            else
            {
                tentativas--;
                Console.WriteLine("Você errou! Tentativas restantes: " + tentativas);
            }
        }
        Console.WriteLine("Você perdeu! O número secreto era: " + numSecreto);
    }
}
